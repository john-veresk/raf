# Project Decisions

- Q: Какой идентификатор проекта нужно показывать в amend-комментариях редактора?
  A: `ID + slug`, то есть имя папки проекта целиком, например `69-amend-whisper`.

- Q: Эту правку ограничить только комментами в amend-шаблоне редактора, или затронуть и другие amend-подсказки?
  A: Ограничить изменение только шаблоном редактора.
